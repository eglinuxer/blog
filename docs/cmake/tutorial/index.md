# CMake 教程

本教程大部分内容来自：[CMake Turorial](https://cmake.org/cmake/help/latest/guide/tutorial/index.html)，加上我自己的理解，总结归纳而成。

## 介绍
CMake 教程提供了一个分步指南，涵盖了 CMake 帮助解决的常见构建系统问题。了解各种主题如何在一个示例项目中一起工作会很有帮助。
